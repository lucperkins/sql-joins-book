# The SQL Joins book

## Git

```shell
git clone https://github.com/lucperkins/sql-joins-book
cd sql-joins-book
```

## Tarball

```shell
mkdir sql-joins-book
curl -L  https://github.com/lucperkins/sql-joins-book/tarball/main | tar --directory sql-joins-book --strip-components=1 -xz
cd sql-joins-book
```
